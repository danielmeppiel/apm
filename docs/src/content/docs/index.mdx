## The missing package manager for AI coding agents

APM brings dependency management to AI agent configuration — the same way npm manages JavaScript packages or pip manages Python packages, APM manages the instructions, skills, prompts, and context that make AI coding agents effective.

## Quick start

```bash
# Install APM
curl -sSL https://raw.githubusercontent.com/microsoft/apm/main/install.sh | bash

# Initialize a project
apm init

# Add a package
apm install microsoft/apm-samples

# Compile to AGENTS.md
apm compile
```

## What APM manages

APM packages can contain any combination of **agent primitives** — the building blocks that configure AI coding agents:

| Primitive | Purpose | Example |
|-----------|---------|---------|
| **Instructions** | Coding standards and guidelines | `.instructions.md` files |
| **Skills** | Reusable capability descriptions | `SKILL.md` files |
| **Prompts** | Parameterized AI task scripts | `.prompt.md` files |
| **Agents** | AI assistant role definitions | `.agent.md` files |
| **Hooks** | Lifecycle event handlers | Hook JSON files |
| **Plugins** | Bundled tool packages | `plugin.json` manifests |
| **MCP Servers** | Model Context Protocol endpoints | MCP configuration |
